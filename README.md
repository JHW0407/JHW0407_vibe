# 🌠 ABOUT

Auracast는  
사용자의 성향과 분위기를 감성적으로 분석하는  
MBTI 테스트 웹사이트입니다 ✨

각 결과마다 다른 이미지와 분위기를 제공하여  
단순한 테스트가 아닌 하나의 무드 경험처럼 느껴질 수 있도록 제작했습니다 🌌

또한 바이브코딩 방식으로  
아이디어를 빠르게 구현하며 완성했습니다 🎧
