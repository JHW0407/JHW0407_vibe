# 🌌 ABOUT PROJECT

## ✨ Auracast란?

Auracast는  
사용자의 분위기와 감성을 분석하는  
오라(Aura) 컨셉의 MBTI 테스트 웹사이트입니다 🔮

단순한 성격 테스트가 아니라  
사용자의 무드와 분위기를 시각적으로 표현하는 데 집중했습니다 🌠

또한 바이브코딩 방식으로  
아이디어를 빠르게 구현하며 제작했습니다 🎧
